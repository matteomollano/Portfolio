It features data preprocessing, SQL database storage, DoS detection using KNN, RandomForest, and IsolationForest models, and a React frontend for visualizations and IP geolocation via MapBox.

Python Scapy Pandas MariaDB scikit-learn React Node.js MapBox

ScannPal

A cloud-based web application vulnerability scanner that automates the process of detecting SQL Injection, Cross-Site Scripting (XSS) and Command Injection vulnerabilities in web applications. It features web page interaction via Java Selenium, a custom web crawling algorithm designed in Java, parallel computing using AWS services, and a web interface to view scan results.

Java Selenium Java AWS SDK EC2 S3 VPC RDS MySQL Node.js EJS
Source Code

Automated Email Alerts

An automated system developed using Python and Google Cloud services to alert team members of when an expensive cloud virtual machine has been running for more than 2 hours. This project helped the cyber team to reduce unnecessary costs for idle GCP instances. Key features include sending email notifications using Python, implementing serverless Cloud Functions, and scheduling the function to run every 30 minutes using Cloud Scheduler.

Python Google Cloud Cloud Functions Compute Engine Google Cloud SDK Python Email Library Cloud Scheduler

Heart Disease Risk Prediction

A data science project that identifies key factors contributing to heart disease risk through analysis of real-world patient data collected from five hospitals (over 900 patients). It features statistical hypothesis testing, dimensionality reduction using PCA and Factor Analysis, and machine learning models including Decision Trees, K-Nearest Neighbors, and Linear Regression for predictive analysis.

R Python Pandas Hypothesis Tests PCA Factor Analysis Machine Learning

Hofstra-Amazon Capture the Flag - 4th Place

Hosted by Amazon and Hofstra University

Finished 4th place in the finals after a first round of 79 teams (14 teams advanced to the finals), competing against 14 finalist teams from 5 universities. Solved challenge problems involving packet capture analysis, Caesar/ROT13/substitution cipher decoding, binary reverse-engineering, steganography, and SQL injection. Utilized tools such as Wireshark for network forensics, Ghidra for binary analysis, Nmap for enumeration and service detection, and sqlmap for remediating SQL injection vulnerabilities.

Senior Project Showcase Competition - 3rd Place

Hosted by Hofstra University

Awarded 3rd place at the Senior Design Poster Showcase for our team's project ScannPal, a cloud-based web application vulnerability scanner. We demonstrated how ScannPal crawls websites, performs SQL and XSS injection tests, and securely stores results in AWS RDS for later analysis. Utilizing parallel computing across 15 EC2 instances, our solution showcases how businesses can proactively identify and address security vulnerabilities in their web applications.
